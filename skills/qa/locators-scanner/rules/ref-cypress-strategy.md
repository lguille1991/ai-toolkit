## Philosophy

Decouple tests from implementation. Use dedicated test attributes that are immune to CSS/JS refactoring. Cypress is CSS-selector-native.

## Locator Hierarchy (first match wins, top to bottom)

| Priority | Strategy | When to use | Code pattern |
|---|---|---|---|
| 1 | `data-cy` attribute | Element has `data-cy` (Cypress's recommended test attribute) | `cy.get('[data-cy="submit-btn"]')` |
| 2 | `data-testid` / `data-test` | Element has any dedicated testing attribute | `cy.get('[data-testid="login-form"]')` |
| 3 | `cy.contains()` | Element identifiable by unique visible text | `cy.contains('button', 'Sign In')` |
| 4 | `id` selector | Element has a stable, unique `id` | `cy.get('#email-input')` |
| 5 | `aria-label` | Element has an `aria-label` attribute | `cy.get('[aria-label="Close dialog"]')` |
| 6 | `name` attribute | Form element with `name` attribute | `cy.get('[name="username"]')` |
| 7 | `role` attribute | Element has explicit `role` | `cy.get('[role="navigation"]')` |
| 8 | CSS selector | Tag + class or attribute combo | `cy.get('button.primary-btn')` |
| 9 | XPath | **Not native** — requires `cypress-xpath` plugin. Avoid. | N/A |

## Framework-Specific Rules

- Cypress strongly recommends adding `data-cy` attributes to source code. When they exist, always prefer them.
- `cy.contains()` is powerful for text matching but fragile if text changes frequently (i18n, A/B tests). Use with caution; note this in comments.
- Never rely on CSS classes alone (`.btn-primary`) — they are tied to styling and will break on CSS refactors.
- For scoping: `cy.get('[data-cy="product-list"]').find('[data-cy="add-to-cart"]')` or `cy.get('[data-cy="product-list"]').contains('Add to Cart')`
- Dynamic classes generated by CSS-in-JS (e.g., `Component_button__3XyZ2`) should be marked as unreliable.

## POM Template (JavaScript)

```javascript
/**
 * {PageName}Page — Page Object Model
 * URL: {url}
 * Generated by locator-extractor skill
 * Framework: Cypress
 * Date: {date}
 */
class {PageName}Page {

  // ── Interactive Elements ──────────────────────────

  /** {comment explaining why this locator was chosen} */
  get {elementName}() {
    return cy.get('{selector}');
  }

  // ── Form Elements ─────────────────────────────────
  // ...

  // ── Content Elements ──────────────────────────────
  // ...

  // ── Structural Elements ───────────────────────────
  // ...

  // ── Actions ───────────────────────────────────────

  navigate() {
    cy.visit('{url}');
  }

  // Add common page actions here
}

export default new {PageName}Page();
```
